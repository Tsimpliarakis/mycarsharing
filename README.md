# Quasar App (quasar)

Run after cloning:
1. npm install
2. npm install @supabase/supabase-js
3. quasar dev --history (or quasar build --history)

Build after cloning:
(npx) quasar build --history
cd /dist/spa
(npx) quasar serve --history
